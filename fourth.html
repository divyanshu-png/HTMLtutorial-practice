1)Div tag (div):

The div tag in HTML defines a division or a section within an HTML document. It is a block-level element, meaning it typically takes up the full available width and starts on a new line. It serves as a generic container for other HTML elements and content, providing a way to group related content together for styling with CSS or manipulation with JavaScript.
Note: The counterpart of block level elements are called inline elements i.e. the elements that take only a specified amount of space

Now let us understand each element here:

1) Form : defines the HTML form
  • Action attribute: Specifies the URL where the form data will be sent upon submission
  • Method attribute: Specifies the HTTP method to use (e.g., get or post).
    a) GET: Appends form data to the URL (visible, limited data). Good for simple queries.
    b) POST: Sends form data in the HTTP request body (not visible, more data). Good for sensitive data(like username password and personal details) or large amounts of data.

2) Input tag: The most versatile element used for inputting the data. Creates a box that can be used for various input types. Has two IMPORTANT attributes; TYPE and PLACEHOLDER or VALUE
